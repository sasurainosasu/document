alma linux 10系　初期設定

//ユーザーの追加
useradd example
passwd example

//sshの接続元を特定のIPのみ許可する
1.IPアドレスの追加
firewall-cmd --permanent --zone=public --add-rich-rule="rule family="ipv4" source address="126.208.149.168" port protocol="tcp" port="22" accept"
firewall-cmd --permanent --zone=public --add-rich-rule="rule family="ipv4" source address="60.79.183.46" port protocol="tcp" port="22" accept"

2.不特定のIPアドレスからアクセスできないようにする
firewall-cmd --remove-service=ssh --zone=public --permanent

3.再起動する
firewall-cmd --reload

※設定が不要になったら
firewall-cmd --zone=public --remove-rich-rule="rule family="ipv4" source address="114.XXX.XXX.72" port protocol="tcp" port="22" accept"

▽設定ファイルはここに置かれている
/etc/firewalld/zones/public.xml

（参考URL：https://blog.netandfield.com/shar/2021/09/almalinux-ssh-ip.html）

//sshd_configの設定
vi /etc/ssh/sshd_config
PermitRootLogin no
systemctl restart sshd

//httpdのインストール
yum install httpd
systemctl start httpd
firewall-cmd --permanent --add-service=http
firewall-cmd --permanent --add-service=https
firewall-cmd --reload
systemctl enable httpd.service

//vsftpdのインストール
yum install vsftpd
firewall-cmd --add-service=ftp
firewall-cmd --add-service=ftp --permanent

//vsftpdを利用可能にする
systemctl enable vsftpd.service
